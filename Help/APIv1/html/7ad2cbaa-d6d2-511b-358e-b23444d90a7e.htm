Properties
  NameDescription

Attach node used for linking with the target part.

A source link type identifier.
guiLinkMode
Mode in which a link between soucre and target is being created.
isLocked
Defines if source must not initiate link requests because there is another source module on the part that has established a link.
linkState
Current state of the source.
linkTarget
Linked target or null if nothing is linked.
nodeTransform
Transform that defines position and orientation of the attach node.
part
Part that owns the source.
