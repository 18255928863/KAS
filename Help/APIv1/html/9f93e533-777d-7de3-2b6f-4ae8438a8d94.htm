public abstract Transform sourceTransform { get; set; }

Property Value

Type: Transform
Remarks
It's not just a position anchor. It's a transform of the game object that represents the joint root model. Its rotation will be adjusted when establishing/updating the link.

This transform is usually located at attachNodePosition and oriented as attachNodeOrientation, but it's not the same transform as the attach node one (due to it can be changed).
