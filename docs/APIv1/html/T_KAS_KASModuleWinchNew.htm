Remarks

The connector is attached to the winch with a cable, and it can link with the compatible link targets. The winch itself is a link source. An EVA kerbal can "grab" the connector and carry it as far as the cable maximum length allows.

Since the winch is a basic link source it can link with the target in the differnt modes. However, it's highly recommended to use the mode TieAnyParts. As it the most flexible, and the winch is capable of changing "docked" vs "non-docked" mode when the link is already made.
