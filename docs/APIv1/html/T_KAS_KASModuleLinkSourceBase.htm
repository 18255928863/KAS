Remarks
This module deals with main logic of linking two parts together. The other party of the link must be aware of the linking porcess. The targets must implement ILinkTarget.

External callers must access methods and properties declared in base classes or interfaces only. Members and methods that are not part of these declarations are not intended for the public use regardless to their visibility level.

Decendand classes may use any members and methods but good practice is restricting the usage to the interfaces and virtuals only.
