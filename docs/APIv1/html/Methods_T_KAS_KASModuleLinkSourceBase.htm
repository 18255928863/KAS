KASModuleLinkSourceBase Methods

The KASModuleLinkSourceBase type exposes the following members.

Methods
  NameDescription
BreakCurrentLink
Breaks the link between the source and the target.
CancelLinking
Cancels linking mode without creating a link.
CheckBasicLinkConditions
Performs a check to ensure that the link between the source and the target, if it's made, will be consistent.
CheckCanLinkTo
Verifies if a link between the parts can be successful.
DecoupleAction
Called when two parts decouple.
GetDrawModulePanelCallback
Returns a method delegate to draw a custom panel.
GetInfo
The return value of this function appears in the part's description in the editor.
(Overrides PartModuleGetInfo.)
GetModuleTitle
Returns module title to show in the editor part's details panel.
GetPrimaryField
Return a string to be displayed in the main information box on the tooltip.
LinkToTarget
Establishes a link between two parts.
LogicalLink
Logically links the source and the target, and starts the renderer.
LogicalUnlink
Logically unlinks the source and the current target, and stops the renderer.
OnAwake
This function gets called only once, during the KSP loading screen. See the Unity documentation on Awake for more information.
(Overrides PartModuleOnAwake.)
OnDestroy
Triggers when Unity object is about to destroy.
OnInitialize
Called on a vessel when it's time to start the physics on it.
(Overrides PartModuleOnInitialize.)
OnKASLinkBrokenEvent
Triggers when a source on the part has broke the link.
OnKASLinkCreatedEvent
Triggers when a source on the part has created a link.
OnLoad
This function is called to initialize the part. The ConfigNode contains the parameters of the module as specified in the part.cfg file, or as you last saved them in OnSave.
(Overrides PartModuleOnLoad(ConfigNode).)
OnPartDie
Triggers when part ois destroyed by the game's logic.
OnPartPack
Triggers when physics stops on the part.
OnPartUnpack
Triggers when physics starts on the part.
OnStart
Called when the flight starts, or when the part is created in the editor. OnStart will be called before OnUpdate or OnFixedUpdate are ever called.
(Overrides PartModuleOnStart(PartModuleStartState).)
OnStateChange
Triggers when a state has been assigned with a value.
PhysicalLink
Links source and target in the physical world.
PhysicalUnlink
Breaks link with the target in the physical world.
RestoreTarget
Finds linked target for the source, and updates the state.
StartLinkGUIMode
Initiates GUI mode, and starts displaying linking process.
StartLinking
Starts the linking mode of this source.
StopLinkGUIMode
Stops any pending GUI mode that displays linking process.
