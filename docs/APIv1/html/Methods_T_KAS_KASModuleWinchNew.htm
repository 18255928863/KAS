Tells if the currently active vessel is an EVA kerbal who carries this winch connector.

(Overrides KASModuleLinkSourceBaseLogicalUnlink(LinkActorType).)

OnLoad
This function is called to initialize the part. The ConfigNode contains the parameters of the module as specified in the part.cfg file, or as you last saved them in OnSave.
(Overrides KASModuleLinkSourceBaseOnLoad(ConfigNode).)
OnPartUnpack
Triggers when physics starts on the part.
(Overrides KASModuleLinkSourceBaseOnPartUnpack.)
OnSave
This function is called when the game is saved to let the part save persistent data. Add any data you want to persist to the ConfigNode. The ConfigNode will then be saved as part of persistent.sfs. When the game is resumed, you can then read this data back out in OnLoad.
(Overrides PartModuleOnSave(ConfigNode).)
OnStartFinished
Notifies that all the modules on the part have been started.
(Overrides PartModuleOnStartFinished(PartModuleStartState).)
PhysicalUnlink
Breaks link with the target in the physical world.
(Overrides KASModuleLinkSourceBasePhysicalUnlink(ILinkTarget).)
ReleaseCableEvent
KSP event.
A context menu item that sets the cable length ot the maximum, and unlocks the connector if it was locked.
ReturnConnectorEvent
KSP event.
Detaches the connector from the kerbal and puts it back to the winch.
ShowMessageForActiveVessel
Shows a message in GUI if the reporting part belongs to the active vessel.
ToggleExtendCableEvent

A context menu item that starts/stops extending the cable.
ToggleRetractCableEvent
