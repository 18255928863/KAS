Remarks
This module only deals with logic part of the linking. It remembers the source and notifies other modules on the part about the link state. The actual work to make the link significant in the game engine must be done by the link source, an implementation of ILinkSource.

External callers must access methods and properties declared in KSP base classes or interfaces only. Members and methods that are not part of these declarations are not intended for the public use regardless to their visibility level.
