Target or null if no valid target was found.
Remarks
Any number of targets can be linked on the part but only one is linked with a particular source. This method goes over all targets on the target part, and returns the one that is linked with the provided source.

It's discouraged to implement this logic in own code since linking approach may change in the future versions.
