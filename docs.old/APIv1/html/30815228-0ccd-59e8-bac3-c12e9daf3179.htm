The ILinkTarget type exposes the following members.

Properties
  NameDescription
attachNode
Attach node used for linking with the source part.
cfgAttachNodeName
Name of the attach node to connect with.
cfgLinkType
A target link type identifier.
isLocked
Defines if target must not accept any link requests because the part is already linked as source.
linkSource
Source that maintains the link or null if nothing is linked.
linkState
Current state of the target.
nodeTransform
Transform that defines position and orientation of the attach node.
part
Part that owns the target.
