The AbstractJointModule type exposes the following members.

Methods
  NameDescription
AdjustJoint
Requests the joint to become unbreakable or normal.
CheckAngleLimitAtSource
Checks if link angle at the source joint is within the limits.
CheckAngleLimitAtTarget
Checks if link angle at the target joint is within the limits.
CheckLengthLimit
Checks if link length is within the limits.
CreateJoint
Sets up a physical link between source and target.
DropJoint
Destroys a physical link between source and target.
DropStockJoint
Destroys stock joint on the part if one exists.
GetClampedBreakingForce
Rounds down the value so what it doesn't contradict with source and target breaking forces.
GetClampedBreakingTorque
Rounds down the value so what it doesn't contradict with source and target breaking torques.
GetDrawModulePanelCallback
Returns a method delegate to draw a custom panel.
GetInfo
The return value of this function appears in the part's description in the editor.
(Overrides PartModuleGetInfo.)
GetModuleTitle
Returns module title to show in the editor part's details panel.
GetPrimaryField
Return a string to be displayed in the main information box on the tooltip.
OnDestroy
Triggers when Unity object is about to destroy.
OnKASLinkBrokenEvent
Triggers when a source on the part has broke the link.
OnKASLinkCreatedEvent
Triggers when a source on the part has created a link.
OnPartPack
Triggers when physics stops on the part.
OnPartUnpack
Triggers when physics starts on the part.
ScaleForceToNode
Scales the force value to the node size.
SetBreakForces
