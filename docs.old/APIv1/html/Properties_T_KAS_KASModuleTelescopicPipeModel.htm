The KASModuleTelescopicPipeModel type exposes the following members.

Properties
  NameDescription
cfgRendererName
Unique name of the randerer that is used by the other modules to find this renderer.
colorOverride
Temporally sets another color to the link meshes.
isLinked
Tells if the source on the part is linked.
isPhysicalCollider
Tells if the link interacts with the rigid bodies.
isStarted
Tells if the renderer is started and active.
linkSource
A link source module that operates on this part. There can be only one.
maxLinkLength
The maximum length to which the telescopic pipe can expand. It's calculated from the model.
minLinkLength
The minimum length to which the telescopic pipe can shrink. It's calculated from the model.
outerPistonDiameter
Diameter of the outer piston. It's calculated from the model.
pistonLength
Length of a single piston. It's calculated from the model.
pistonPrefab
Prefab for the piston models.
pistons
Pistons that form the strut pipe.
shaderNameOverride
Temporally sets another shader to the link meshes.
sourceTransform
Base position/direction of the connection point at the beginning of the link. The source joint models will be aligned against this transform.
srcJointHandleLength
Distance of the source part joint pivot from it's base. It's calculated from the model.
srcPartJoint
Model that represents a joint at the source part.
srcPartJointPivot
Pivot axis object on the source joint.
srcStrutJoint
Model at the pipe's start that connects to the source joint pivot.
stretchRatio
Defines how significantly the link has stretched or shrinked comparing to it's "normal" state.
targetTransform
Base position/direction of the connection point at the end of the link. The target joint models will be aligned against this transform.
trgJointHandleLength
Distance of the target part joint pivot from it's base. It's calculated from the model.
trgStrutJoint
Model at the pipe's end that connects to the target joint pivot.
trgStrutJointPivot
Pivot axis object at the pipe's end.
